Mi az a HTML?

  • A HTML (HyperText Markup Language) egy leíró nyelv, amelyet weboldalak készítéséhez használunk. Egy HTML kód egyértelműen meghatározza a weboldal felépítését, kinézetét. Azaz a HTML nyelven megírt kódot a webböngésző értelmezi, majd a kód alapján megjeleníti a már általunk megszokott oldalakat.
  • A HTML úgynevezett 'tag'-ekből épül fel, ami az adott elem alap tulajdonságait írja le a böngészőnek
  • Akit bővebben érdekel a téma, sok olvasnivalót talál a W3schools oldalán.

Mi az a CSS?

  • A CSS (azaz „egymásba ágyazott stíluslapok”) a webhelyek vizuális stílusát határozza meg.
  • A CSS segítségével egy weboldal HTML elemeinek különböző paramétereket szabhatunk meg. Ezek a paraméterek kihathatnak az elemek színére, méretére, és egyéb vizuális tulajdonságaira.
  • A külső CSS egy külön fájl, amire a HTML weblap hivatkozik. A belső CSS a HTML dokumentum fejrészében helyezkedik el. az Inline CSS a HTML dokumentum szövegében közvetlenül helyezkedik el.

Mi az a JavaScript?

  • A JS egy magas szintű, JIT, objektum orientált programnyelv a webre fejlesztve.
  • A JS csinál mindent, ami dinamikus egy weboldalon.
  • A JS a böngésző motorjában fut egy saját környezetben, nem éri el a saját gépünket.
